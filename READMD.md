1. Install the packages by npm i
2. Provide the Below Data In Env file (I used mongo db)
   DB_USER_NAME=
   DB_PASSWORD=
   DB_DATABASE_NAME=
   DB_PORT=
   DB_HOST=
   TICKETS_TO_SEED_COUNT=
3. Watch - by npc tsc -w
4. Run - npm run start
5. During start it self seed will happen so no need to run separatlely.
